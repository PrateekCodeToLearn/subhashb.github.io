VIT Programming Challenge - Nov 2016

  1. Below is a slightly altered version of the Human family tree, as per existing fossil records. Write a program to find the closest human ancestor given a pair of fossils.

  2. Given two sequence of Alphabets, alphabetically order them, count all possible combinations occurring in alphabetical order while alternating between the sequences and print the longest combination. Note that the first letter of the combination has to be from the first sequence.

    Example:
     
    Sequence 1: {'Z', 'B', 'G'}
    Sequence 2: {'A', 'H', 'K', 'C', 'M'}
     
    Alphabetically ordered Sequence 1 = {'B', 'G', 'Z'}
    Alphabetically ordered Sequence 2 = {'A', 'C', 'H', 'K', 'M'}
     
    Possible Combinations:
      B C
      B C G
      B C G H
      B C G H Z
      B H
      B H Z
      B K
      B K Z
      B M
      B M Z
      G H
      G H Z
      G K
      G K Z
      G M
      G M Z
     
    Count of Possible Combinations: 16
    Longest Combination: B C G H Z

  3. Write a function that, given a list of integers (both positive and negative) returns the product of the contiguous subsequence with maximum product. Thus, given the sequence [2,3,-2,4] it should return 6 (2*3)

    Example 1:
     
      Input: {1, -2, 3, 0, 4, -5}
      Output: 4
     
    Example 2:
     
      Input: {1, -2, 3, 4, -5, 0, 2, 5, 1, 0, 8, -3, 1, 0, 0, 1, 3, -5, 7, -6, 2, 2}
      Output: 2520
  4. If the numbers 1 to 5 are written out in words: one, two, three, four, five, then there are 3 + 3 + 5 + 4 + 4 = 19 letters used in total.

    For a given input (between 1 to 99999 inclusive), count the number of letters that would be used.

    Example 1:
     
      Input: 10
      Output: 39
     
    Example 2:
     
      Input: 12345
      Output: 400954

    NOTE: Do not count spaces or hyphens. For example, 342 (three hundred and forty-two) contains 23 letters and 115 (one hundred and fifteen) contains 20 letters. The use of "and" when writing out numbers is in compliance with British usage.
